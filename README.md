# 📃 Dynamic To-Do Web Application

A clean, responsive, and minimal single-page To-Do List web application designed to track tasks. Developed from scratch as part of the frontend engineering and data persistence evaluation.


## 🛠️ Tech Stack Used

* **HTML5:** Semantic workspace layout blocks (`.cont`, `.input-cont`).
* **CSS3:** Native styling, linear gradients (`#9fac2b` to `#81b622`), flexbox alignments.
* **Vanilla JS:** Native DOM manipulation, click trackers, state serialization.
* **Storage Engine:** 100% native Web Storage API (LocalStorage).


## ✨ Key Features Included

* **Instant Appending:** Fast task addition via dynamic layout injection.
* **Persistent Memory:** LocalStorage integration preserves data across restarts.
* **Toggle Mechanics:** Built-in strike-through triggers (`.chkd`) for quick checking.
* **Validation Shield:** Active security layer blocks empty field inputs.
* **Dynamic Deletion:** Individual item removal using absolute-positioned targets (`×`).


## 💻 Local Workspace Startup

1. **Clone** this project folder to your local system directory.
2. Launch `index.html` inside any standard web browser engine.
